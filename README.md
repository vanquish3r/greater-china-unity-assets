# Greater China Unity Asset Archive

**[View the Live Website Here](https://vanquish3r.github.io/greater-china-unity-assets/)**

## Purpose

Unity recently announced that assets from publishers located in the Greater China region (including China, Hong Kong, and Macau) will be delisted from the global Unity Asset Store. This goes into effect on **March 31st, 2026**.

This repository serves as an archive of the roughly 2,700 assets that will be affected by this change. It provides a simple, searchable, and filterable web interface for developers to discover and acquire these assets before the deadline. 

For developers who have already acquired these assets, they will remain available and functional. However, publishers will no longer be able to provide updates or support through the global Asset Store after the deadline.

**[Read the official announcement here](https://discussions.unity.com/t/a-notice-for-asset-store-assets-from-publishers-in-greater-china/1711108)**

## Data Source

The asset data is currently compiled into a CSV file (`Assets_being_removed_March_31st.csv`) and converted/stored as JSON (`data.json`) for the web application to consume quickly.

**[Original PDF list of the affected assets](https://cdn.bfldr.com/S5BC9Y64/as/wpcqrs7fxjcwgqm9m6qz/Assets_being_removed_March_31st)**
